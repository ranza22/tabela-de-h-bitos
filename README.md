# 📊 Tabela de Hábitos

Projeto de controle de hábitos desenvolvido com JavaScript, com sistema de streak, edição e salvamento no navegador (localStorage).

---

## ⚙️ Funcionalidades
- Sistema de streak (sequência de hábitos)
- Edição de hábitos
- Salvamento automático com localStorage
- Interface interativa

---

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

---

## 🧠 Aprendizados
Durante o desenvolvimento deste projeto, pratiquei lógica de programação, manipulação do DOM e uso de armazenamento local (localStorage).  
Também enfrentei desafios relacionados à organização do código e resolução de erros, o que contribuiu para minha evolução prática.
